

## 项目结构

```
education-platform/
├── pom.xml                    # 父POM
├── platform-common/           # 公共模块
├── platform-gateway/          # 网关服务 (8080)
├── user-service/              # 用户服务 (8081)
├── resource-service/          # 资源服务 (8082)
├── course-service/            # 课程服务 (8083)
├── community-service/         # 社区服务 (8084)
├── report-service/            # 报告服务 (8085)
└── audit-service/             # 审核服务 (8086)
```
